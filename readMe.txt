1.总体目标：
	输入		规划、调度算法

	条件	1	自动随机事件			手工事件
	条件2	自动调度				手工调度
	条件3				底层资源


	输出		调度效果评价
	（反馈回输入）

2.无人车、调度、模拟
	无人车是装逼用的噱头

	调度主要有路径规划和任务调度

	模拟主要有
			地图要素
			随机地图事件（损坏，拥堵）、随机任务事件、随机车辆事件
			资源管理：站点，车辆，资源，调度员、维护员、用户

3.地图模拟
	利用leaflet构建地图，利用json形式的数据作为输入
	以交大本部和东二环以西为基础建模

4.模块划分
	随机模块（一定要和其他模块分离开来）
	地图模块（属于被调用的资源）
	任务模块（属于被调用的资源）
	管理模块（CRUD干起来）
	无人车模块（交互通信是关键）
